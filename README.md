# DeliveringDelights
Food Delivery Mobile App.

# Features
* Proactive Notifications
* Self-Service Problem Reporting 
* Order Tracking
* User Feedback and Rating System

# Course Overview
* Basic React Native Components
* Database Design using Appwrite
* User Registration & Authenticaiton
* Deployment

# Tech Stack
* React Native
* AppWrite
* HTML, CSS and JS
